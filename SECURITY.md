# Security Policy

## Maintainer

Tyvoll Pvt. Ltd.

## Reporting a Vulnerability

Please do not disclose security vulnerabilities publicly through GitHub issues.

Report security concerns privately to:

- contact@tyvoll.com

Please include:

- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation or fix (if known)

We will acknowledge valid reports and work to investigate and remediate issues as quickly as possible.

## Supported Versions

The main branch is considered the supported version of this project.

## Sensitive Information

Do not commit or disclose:

- API keys
- Access tokens
- Passwords
- Tunnel credentials
- .env files
- Local configuration containing secrets
- Generated logs containing sensitive information

## Responsible Disclosure

Please provide a reasonable amount of time for investigation and remediation before public disclosure of a security issue.
